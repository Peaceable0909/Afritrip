# NIGERIA wordmark images

These two files control what appears **inside the giant NIGERIA letters** on the
homepage — independent of the hero background.

| File | What it does |
|---|---|
| `fill.jpg` | The image visible inside the letters. Use something wide (1600px+) with strong color — sunsets, landscapes, textiles all work. |
| `texture.jpg` | The grunge/scratch overlay blended on top for the weathered paint look. Black & white, high contrast works best. |

**To swap:** replace either file, keep the same name, refresh. No code changes.
